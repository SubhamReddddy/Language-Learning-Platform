# 🌍 Language Learning Platform

🚀 **Live Demo**: [Click Here](https://language-learning-platform-topaz.vercel.app/)  

## 📌 Overview
The **Language Learning Platform** is an interactive web application designed to help users learn new languages efficiently. It offers engaging lessons, vocabulary exercises.

## ✨ Features 
✅ **Interactive Lessons** – Engaging language courses  
✅ **Vocabulary Builder** – Learn and review words with flashcards  
✅ **Responsive UI** – User-friendly UI 

## 🛠️ Tech Stack
| Category     | Technology Used |
|-------------|----------------|
| **Frontend** | React.js, Vite, Tailwind CSS |
| **Backend**  | Node.js, Express.js |
| **Database** | MongoDB |
| **Authentication** | JWT (JSON Web Token) |
| **Hosting**  | Vercel |

## 🚀 Installation Guide
1️⃣ **Clone the Repository**  
```sh
git clone https://github.com/your-username/language-learning-platform.git
cd language-learning-platform
```
2️⃣ **Install Dependencies**  
```sh
npm install
```
3️⃣ **Run the Application**  
```sh
npm run dev
```
🔹 The app will be available at `http://localhost:5173`
